# How to build



- Install [rust](https://www.rust-lang.org/learn/get-started)

- Open Terminal / PowerShell / cmd

- Clone this repo

  ```
  git clone https://github.com/unknown-marketwizards/tradinghero-desktop.git
  ```

- Enter folder

  ```
  cd tradinghero-desktop
  ```

- Build

  ```
  cargo build --release
  ```
